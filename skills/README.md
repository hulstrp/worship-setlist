# WorshipFlow 개발 스킬

이 디렉토리에는 WorshipFlow 프로젝트 개발 시 유용한 스킬과 패턴이 정리되어 있습니다.

## 스킬 목록

| 파일 | 설명 |
|------|------|
| [youtube-ccm-links.md](youtube-ccm-links.md) | 한국 CCM YouTube 링크 검증 및 관리 |
| [fastapi-auth-patterns.md](fastapi-auth-patterns.md) | FastAPI 인증 패턴 (MVP vs 프로덕션) |
| [worshipflow-db-management.md](worshipflow-db-management.md) | 찬양 DB 관리 및 쿼리 패턴 |

## 사용법

각 스킬 문서에는:
- 배경 설명
- 코드 예시
- 주의사항
- 관련 파일 경로

가 포함되어 있어 해당 작업 시 참고할 수 있습니다.

## 스킬 추가 방법

새로운 스킬을 추가하려면:

1. `skills/` 디렉토리에 마크다운 파일 생성
2. 제목, 설명, 코드 예시, 주의사항 포함
3. 이 README에 링크 추가
